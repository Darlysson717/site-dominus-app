# Dominus — Site Estático (Clone simplificado)

Este repositório contém uma versão estática simplificada do site apresentado em:

https://dominus-veiculos-bgfplzr.gamma.site/#

O objetivo é reproduzir o layout e as seções principais (hero, recursos, como funciona, inventário, contato).

Como rodar localmente (PowerShell):

```powershell
cd 'c:\Users\darly\Desktop\site dominus app'
python -m http.server 8000
# Abra http://localhost:8000 no navegador
```
